# 文件结构

* 小程序主要包含两部分文件:
    - `app`: 描述程序整体
    - `page`: 描述各个页面
* app部分由3个文件组成, 必须放在项目根目录
    - `app.js`: 必填, 小程序逻辑
    - `app.json`: 必填, 小程序公共设置
    - `app.wxss`: 非必填, 小程序公共样式表
* page部分由4个文件组成:
    - `js`: 必填, 页面逻辑
    - `wxml`: 必填, 页面结构
    - `wxss`: 非必填, 页面样式表
    - `json`: 非必填, 页面配置

**注意:** 描述页面的4个文件必须具有相同`路径`和`文件名`
